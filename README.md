### 个人博客网站

## 技术栈

* vue 3.2.45
* pinia 2.0.13
* vue-router 4
* axios 0.26.1
* vanta 0.5.22
* wow.js 1.2.2
* vite 2.9.0
* typescript 4.5.4
* naive-ui 2.34.3

## 启动

* 测试环境
  `yarn dev`

* 线上环境
  `yarn start`

## 部署

* 打包命令
  `yarn build`
